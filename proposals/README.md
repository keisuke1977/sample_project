# プロダクト提案

## 元のアイデア
大手企業に勤める女性従業員を対象に、健康経営・女性活躍推進の文脈でB2B向けの女性健康管理アプリを作りたい。
毎日のチェックで「気づき」を与え、コンテンツで「学び」を促し、医療専門家への「相談」につなぐ3ステップのフロー。
法人契約で企業が導入し、従業員が無料で利用する形態。

## 提案一覧

| 提案 | コンセプト | 特徴 | 向いている人 |
|------|------------|------|--------------|
| A | 毎日1分チェックイン特化型 | シンプルな継続設計。まず習慣化を最優先 | 「まず使ってもらえるか」を検証したい |
| B | 健康経営DXプラットフォーム型 | 従業員アプリ＋人事向けダッシュボードの二刀流 | 企業の健康経営レポートまで一気通貫にしたい |
| C | 専門家コネクト特化型 | 産婦人科・看護師などへのオンライン相談を核に据える | 「医療へのアクセス改善」を最大の価値にしたい |

## 次のステップ

2. 「Aがいい」「BとCを組み合わせたい」など教えてください
