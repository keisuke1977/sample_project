## 機能詳細（主要機能）

### 毎日の体調チェック
**概要：** 毎朝1分以内で完了するタップ式体調チェック

**ユーザーストーリー：**
「忙しい朝でも、タップするだけで自分の体調を記録したい。なぜなら続けないと意味がないから。」

**入力：**
- 睡眠の質（5段階タップ）
- 身体の疲れ（5段階タップ）
- 気分・メンタル（5段階タップ）
- 気になる症状（複数選択）
- 月経状況（3択）

**出力：**
- 「今日の気づき」メッセージ
- 関連コンテンツのレコメンド
- 体調スコア（グラフ用データとして蓄積）

**受け入れ条件：**
- [ ] 回答から送信完了まで1分以内
- [ ] フィードバックメッセージは入力内容に応じてパーソナライズされる
- [ ] 送信後に関連コンテンツが1〜3件表示される

### チャット相談（看護師・助産師）
**概要：** 気軽にテキストで相談できる、最初の専門家接触窓口

**ユーザーストーリー：**
「受診するほどでもないかもしれないけど、この症状が気になる。専門家に手軽に聞きたい。」

**入力：**
- 相談カテゴリ選択
- 症状・状況のテキスト入力
- 体調チェック履歴（自動連携・任意）

**出力：**
- 看護師・助産師からの返信メッセージ
- 受診推奨の場合：産婦人科医相談への案内

**受け入れ条件：**
- [ ] 相談送信後24時間以内に初回返信がある
- [ ] 「診断」「処方」に相当する表現は使用しない
- [ ] 相談履歴はユーザー自身のみ閲覧可能（企業側には非公開）
