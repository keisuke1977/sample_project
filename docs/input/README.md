# Femcare（仮）- 企業向け女性健康管理プラットフォーム

## 概要
大手企業に勤める女性従業員の健康を、「気づき→学び→相談」の3ステップでサポートするB2B向け女性健康管理Webアプリ。健康経営・女性活躍推進の文脈で企業が導入し、従業員がブラウザから無料で利用する。

## 解決する課題
女性特有の体調変化（月経・PMS・更年期など）が見過ごされ、不調が積み重なって離職・休職につながっている。婦人科受診のハードルが高く、「相談できる場所がない」状況を解消する。

## ターゲットユーザー
- **従業員側：** 20〜50代の大手企業勤務女性
- **企業側：** 人事部・健康経営推進担当・産業医チーム

## 主要機能
- 毎日の体調チェック＋パーソナライズされた「気づき」フィードバック
- 女性ライフステージ別（月経・妊活・更年期）コンテンツ
- 看護師・助産師へのチャット相談 ＋ 産婦人科医とのオンライン相談
- 人事向け匿名集計ダッシュボード＋健康経営レポート自動生成
