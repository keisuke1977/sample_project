# ターゲットユーザー

## メインペルソナ①：従業員ユーザー（30代）

### 基本情報
- 名前（仮）：田中 さやか
- 年齢：32歳
- 職業：大手メーカー 営業職
- 生活環境：都内勤務、既婚、子なし。フルタイム勤務で残業も多い

### 1日の流れ
- 7:00 起床、朝食・身支度
- 8:30 出社、メールチェック・営業準備
- 12:00 昼食（デスクで済ませることも多い）
- 18:00〜20:00 退社
- **→ 朝の通勤中・昼休みにスマホを触る時間がある**

### 課題・ペイン
- 月経前になるとイライラ・集中力低下が起きるが、「気のせい」と思って我慢している
- 生理痛がつらい日も「みんなそんなもの」と思って鎮痛剤で乗り切る
- 婦人科に行くべきか判断できず、受診を先延ばしにしている
- 職場の産業医に相談したいが「会社に知られそうで怖い」

### 現在の解決方法
- 市販の鎮痛剤・漢方を自己判断で服用
- Googleで症状を検索するが信頼できる情報か判断できない
- ルナルナで月経日を管理しているが、それ以上は使いこなせていない

### 求めているもの
- 自分の体調パターンを客観的に把握したい
- 「これって受診が必要？」を気軽に聞ける場所
- 忙しい中でもサッと使える手軽さ

### このプロダクトを使う瞬間
- 朝のチェックイン（通勤中・朝食時）
- 体調が気になった時に専門家に相談
- 月経前後のコンテンツを読んで対処法を学ぶ

### 情報収集の方法
- Instagram（健康・美容系アカウント）
- Google検索
- 職場の同僚からの口コミ

## メインペルソナ②：従業員ユーザー（40代）

### 基本情報
- 名前（仮）：鈴木 恵子
- 年齢：47歳
- 職業：大手商社 管理職（部長）
- 生活環境：都内勤務、既婚、子供2人（高校生）。チームをマネジメントする立場

### 課題・ペイン
- 最近、急な動悸・ほてり・睡眠障害が続いているが更年期とは思いたくない
- 管理職として「体調不良を見せられない」というプレッシャーがある
- 忙しくて婦人科に行く時間がない
- 同世代の同僚に相談できるが、正確な情報なのか不安

### このプロダクトを使う瞬間
- 気になる症状が出た時に専門家に確認
- 更年期に関するコンテンツで自己理解を深める
- 月次の体調傾向を振り返る

## メインペルソナ③：企業側ユーザー（人事担当）

### 基本情報
- 名前（仮）：山本 健一
- 年齢：43歳
- 職業：大手企業 人事部 健康経営推進担当
- 役割：健康経営優良法人の認定取得・維持、従業員の健康施策の企画・運営

### 課題・ペイン
- 女性従業員の健康課題を可視化できていない
- 健康経営レポートの作成が手間で、データ収集に時間がかかる
- 「女性活躍推進」の施策として何をすべきか具体的なアイデアが不足
- 導入したツールが従業員に使われずに終わる失敗経験がある

### 求めているもの
- 経営層・健保に提出できるレポートが自動で作れること
- 従業員に使ってもらえる（継続率が高い）ツール
- 健康経営優良法人の認定要件を満たすエビデンス

### このプロダクトを使う瞬間
- 月次・四半期でダッシュボードにログインしてレポート出力
- 導入効果を経営会議で報告する際のデータ参照

## 非ターゲット

- 男性従業員（将来的な拡張は検討余地あり）
- 医療機関（ユーザーではなく連携パートナー）
