# ユーザーフロー

## 【従業員アプリ】

### 1. オンボーディングフロー

#### 初回登録
1. 企業から招待リンク（メール）を受け取る
2. リンクをクリックしてブラウザでWebアプリにアクセス
3. 招待コードを入力してアカウント作成（メールアドレス＋パスワード）
4. プロフィール設定（年代・ライフステージ：月経あり / 妊活中 / 産後 / 更年期等）
5. 月経周期の初期設定（任意）
6. チェックイン通知の時間設定
7. プライバシーポリシー・同意（「あなたの個人データは会社に共有されません」を明示）
8. ホーム画面へ

#### 初回利用
1. チュートリアル（スキップ可）：「毎朝1分チェックインするだけ」の説明
2. 最初のチェックイン体験
3. フィードバック表示 →「続けると体のパターンが見えてきます」

---

### 2. メインフロー：毎日のチェックイン

**目的：** 体調の変化に気づき、パーソナライズされたフィードバックを得る

**フロー：**
1. 朝、Web Push通知またはメール通知が届く「今日の体調チェックをしましょう」
2. 通知をタップ / クリック → ブラウザでチェック画面が開く
3. タップ式で回答（5〜7問）
   - 睡眠の質（5段階）
   - 身体の疲れ（5段階）
   - 気分・メンタル（5段階）
   - 気になる症状（複数選択：頭痛 / 腹痛 / むくみ / ほてり 等）
   - 月経状況（今日は月経中？ / 月経前？ / 通常期？）
4. 「今日の気づき」が表示される
   - 例：「月経前の時期です。イライラや集中力の低下はPMSのサインかもしれません」
5. 関連コンテンツのおすすめ表示
6. ホームに戻る

**成功条件：** 1分以内にチェックインが完了し、気づきメッセージが表示される

---

### 3. メインフロー：コンテンツで学ぶ

**目的：** 自分の体の変化を理解し、セルフケアに活かす

**フロー：**
1. ホーム画面の「今日のコンテンツ」をタップ
2. カテゴリ一覧（月経ケア / PMS / 更年期 / 妊活 / メンタルヘルス 等）
3. コンテンツを選択（記事 or 短動画）
4. 読了 → 「役に立った？」フィードバック
5. 関連コンテンツのおすすめ表示

---

### 4. メインフロー：専門家に相談する

**目的：** 「受診すべきか」「この症状は何？」を専門家に確認する

#### チャット相談（看護師・助産師）
1. ホーム or 気づきメッセージから「相談する」をタップ
2. 相談カテゴリを選択（月経 / PMS / 更年期 / 妊活 / その他）
3. 症状・体調の概要を入力（テキスト）
4. 担当看護師・助産師がアサインされる（数時間〜翌日）
5. チャットでやり取り
6. 「受診が必要」と判断された場合 → 産婦人科医相談への案内

#### オンライン相談（産婦人科医）
1. 「医師に相談する」をタップ
2. 空き枠のカレンダーから予約（テキスト相談 or ビデオ相談を選択）
3. 事前に症状・体調履歴を共有（アプリのデータを自動連携）
4. 相談実施（テキスト or ビデオ）
5. 相談後メモ・受診アドバイスが記録される
6. 必要に応じて近隣の婦人科クリニックを案内（受診サポート）

---

### 5. サブフロー：体調履歴を振り返る

1. ホーム → 「体調記録」タブをタップ
2. 月間カレンダー表示（体調スコアが色で表示）
3. 特定の日をタップ → その日のチェック内容を確認
4. グラフ表示（睡眠・気分・身体症状の推移）

---

## 【企業・人事向けダッシュボード】

### 1. 初回セットアップ

1. 契約完了後、管理者アカウントを発行
2. 企業情報・部署情報を登録
3. 従業員への招待メール一括送信
4. 初期ダッシュボード確認

### 2. メインフロー：月次レポート確認

**目的：** 女性従業員の健康傾向を把握し、施策に活かす

1. ダッシュボードにログイン
2. 月次サマリーを確認
   - アクティブ率（利用継続率）
   - 体調傾向スコア（全体・部署別・年代別）
   - よく報告される症状トップ5
   - 専門家相談利用率
3. アラート確認（体調スコアが低い傾向のグループ → 匿名）
4. 月次レポートPDFをダウンロード

### 3. メインフロー：健康経営レポート出力

1. 「レポート出力」メニューへ
2. 期間・フォーマットを選択（健康経営優良法人 / 健保提出 / 社内用）
3. 自動生成 → プレビュー確認
4. PDFダウンロード
